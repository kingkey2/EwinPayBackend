加密方式

加密方式皆为SHA256

加密后需再转为大写

php:

        
                hash('sha256', $data, false);

java:

        
                MessageDigest digest = MessageDigest.getInstance("SHA-256");
                digest.reset();
                digest.update(input.getBytes("utf8"));
                toReturn = String.format("%064x", new BigInteger(1, digest.digest()));

加密字串:

5566520

加密后结果:

69481E4583F7A8FBC1200BC629603CB9A0D5B16822DE87B0ACA965C89CD80F00

线上测试网址:https://emn178.github.io/online-tools/sha256.html

相关 API

支付申请

URL: {APIURL}/RequirePayingUrl

Method: Post

Content-Type: application/json

Request :

PS : 签章使用SHA256加密。

Parameter Type Required Description
ManageCode String 是 商户代码
Currency String 是    币别(详见对照表)
Service String 是    服务类型(详见对照表)
CustomerIP String 是 提单会员IP
OrderID String 是 订单编号自行产生后带入
OrderDate String 是 订单日期
字串格式要求为 yyyy-MM-dd HH:mm:ss ex:2018-01-01 11:22:33
OrderAmount Number 是 商户订单金额
只支援整数
RevolveURL String 是 回传网址
UserName String 否 汇款人户名
如果为实名制渠道需填写
State String 否 公用回传参数
商户如果支付请求时传递了该参数，则通知商户支付成功时会原样回传该参数(允许值为空字符串，如： State="")
Sign String 是 签章:需再使用SHA256加密后转大写
OrderAmount如有小数时请将尾数0去除,100.00请带入100 (ex:100.00->100) 后加入签名中
格式:ManageCode=商户代码&Currency=币别&Service=服务类型&OrderID=订单编号&OrderAmount=商户订单金额&OrderDate=订单日期&CompanyKey=金钥 
            

                {
                    ManageCode:'ManageCode',
                    Currency:'CNY',
                    Service:'AA001',
                    CustomerIP:'111.111.11.111',
                    OrderID:'OID001',
                    OrderDate:'2020-01-01 01:01:01',
                    OrderAmount:100.00,
                    RevolveURL:'http://ReturnURL',
                    Sign:'ManageCode=ManageCode&Currency=CNY&Service=AA001&OrderID=OID001&OrderAmount=100&OrderDate=2020-01-01 01:01:01&CompanyKey=CompanyKey'
                }

Response

Parameter Type Description
PayingSerial String 系统订单号
Status Number 0=申请成功
1=申请失败
Message String 其他资讯
OrderAmount Number 金额
Url String 支付网址 
            
                {
                    "PayingSerial":"PT2019122212480000000575",
                    "OrderAmount":100.0000,
                    "Status":0,
                    "Message":null,
                    "Url":"Url"
                }
